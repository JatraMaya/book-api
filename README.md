# RestAPI example project with FastAPI

## How to use
* install the necessary package / dependecies e.g fastapi sqlmodel etc

- with uv
```bash
	$ cd book-api
	$ uv sync
```
-with pip
```bash using pip
	$ cd book-api
	$ python -m venv .venv
	$ source .venv/bin/activate
	$ pip install -r requirement.txt
```
* Copy the env_example to a new .env file
```bash
	$ cp env_example .env
```
